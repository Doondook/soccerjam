Soccerjam
=========

Counter-Strike Soccer Mod

1. Extract SoccerJam+.zip to your cstrike folder.
2. Install GeoIP+ module by Arkshine: https://forums.alliedmods.net/showthread.php?t=95665
3. Enjoy.
